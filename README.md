# 🎬 Movie Dekho - Online Movie Ticket Booking WebApp 🎟️

## 🚀 Overview
**Movie Dekho** is an innovative and easy-to-use movie ticket booking platform, designed to streamline the cinema experience for users. Built during the [**Stack Hack 2.0 Hackathon**](https://xathon.mettl.com/event/stackhack-2), this web app ensures hassle-free seat selection, booking, and payment processing—all in a few clicks. 🏆 Our team **Ctrl+Alt+Defeat** proudly secured **27th position** out of **12000 participants**.

## 🛠️ Tech Stack
- **Frontend**: Tailwind CSS, Bootstrap, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Other Tools**: Integrated secure payment gateway for a smooth booking process.

## 📝 Backend Setup
1. **Set up your PHP environment**.
2. **Import the MySQL database** (provided in the `/db` folder).
3. **Update the database credentials** in the backend configuration files.

## 📝 Frontend Setup
1. **Open the project folder**.
2. **Run the project** via a local server or host it on a web hosting platform.

## 📝 Access Admin Panel
- Login as an admin to manage movies, showtimes, and bookings.

## 📝 Start Booking
- Users can sign up, browse movies, select seats, and complete bookings through the payment gateway.

## 📊 Project Features
### 🧑‍💼 Admin:
- Register cinema information, manage showtimes, and approve bookings.
- Simple UI to make management tasks easier.

### 👤 Customer:
- Browse movies, view showtimes, select preferred seats, and pay securely.
- View booking history, rate movies, and give feedback.
